# Data Warehouse Ventas

## Descripción
Este proyecto consiste en la creación de un Data Warehouse utilizando un modelo dimensional tipo estrella.

## Estructura
- DimCliente
- DimProducto
- DimTiempo
- FactVentas

## Proceso ETL
Se realizó extracción desde tablas OLTP (Cliente, Producto, Factura), transformación de datos y carga hacia las dimensiones y tabla de hechos.

## Tecnologías
- SQL Server
